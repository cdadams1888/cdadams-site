Developer⇩

I can remember tearing computers apart, putting them back together, and trying to figure out what exactly was going on with the command line. I couldn't have been any older than 8, and the computer that I used was already ancient and went by the name of Tandy!

My passion for computers continued to persist throughout elementary and middle school when dial up internet was introduced, and I started to become obsessed with downloading music, burning cd's for friends and family, gaming, and building and repairing my own computers.

Throughout high school I continued to spend a good deal of time engaged with the machines that I've built, but knew nothing aside from the hardware components that I fiddled around with. It was when I enrolled in an introductory Python programming course at Saint Petersburg College that I discovered a true passion for coding. I was hooked in from the start, and I started to think like a machine!

I'm an evolving Novice Web Developer; I know HTML5, CSS3, JavaScript, Python, Photoshop, and Responsive Web Design using CSS3 Media Queries.

I'm currently furthering my knowledge in JavaScript, JQUERY, Bootstrap, Git / Github, and many other exciting languages and tools! This site was hand coded by me, and is being used as a means for learning & experimentation.
